# PPTyping

PPTyping is a typing game with a minimal curriculum and fan-fiction story paths
inspired by Pokemon and My Little Pony characters.

## Status

This is an unofficial fan project and remix. It is not endorsed by, sponsored
by, or officially licensed by Pokemon, The Pokemon Company, Nintendo, Hasbro, or
the My Little Pony rights holders.

The project is intended as a non-commercial, hopefully fair-use fan work.
Nothing in this repository should be read as a claim of ownership over those
underlying franchises.

## Run the game

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
